WORKFLOW FEATURES
================================================================================

Purchase Order Lifecycle (8 Statuses):
Draft
  ‚Üí Submitted (Submit for approval)
  ‚Üí Confirmed (Approved by manager)
  ‚Üí In Transit
  ‚Üí Delivered (Ownership TRANSFERS)
  ‚Üí Inspected (Inspection passed)
  ‚Üí Stocked (Auto-inventory update)
  OR Cancelled (Any point)

Ownership Transfer (3 Stages):
1. PENDING
   - Order created
   - Ownership with supplier
   - Status: Draft-Confirmed

2. TRANSFERRED
   - Goods delivered
   - Ownership CHANGES to buyer
   - Status: Delivered

3. CONFIRMED
   - Inspection passed
   - Stock-in complete
   - Ownership confirmed in inventory
   - Status: Stocked

Inspection Workflow:
Pending ‚Üí In Progress ‚Üí Passed/Failed/Partial
- Per-item inspection
- Acceptance/rejection tracking
- Can't stock without PASSED status
- Blocks operations if failed

Automatic Stock-In (One API Call):
1. Creates/updates RoundWoodInventory
2. Creates RoundWoodStockTransaction
3. Updates quantities and costs
4. Calculates average cost
5. Sets ownership to CONFIRMED
6. Creates procurement log entry

Cost Tracking:
- Item subtotal = volume √ó unit_cost
- PO total = sum of items
- Inventory average = total_cost / total_volume
- Every transaction logged

QUICK START STEPS

Step 1: Create Wood Types

  Save

Step 2: Create Purchase Order
  Admin ‚Üí Round Wood ‚Üí Purchase Orders ‚Üí Add

  Save

Step 3: Add Items
  In PO form ‚Üí Items section
  Click "Add another item"
  Fill: Wood Type, Quantity, Diameter, Length, Cost
  Save (volume calculates automatically)

Step 4: Submit & Confirm
  Open PO ‚Üí Status dropdown ‚Üí Submitted
  Admin approves ‚Üí Status ‚Üí Confirmed

Step 5: Mark Delivered
  Admin ‚Üí Round Wood ‚Üí Purchase Orders
  Open PO ‚Üí Status ‚Üí Delivered
  Ownership automatically transfers

Step 6: Inspect
  Same PO ‚Üí Inspection Status ‚Üí In Progress
  Inspect each item
  Mark as Passed

Step 7: Stock In
  Same PO ‚Üí Status ‚Üí Stock In button
  Automatically:
  - Creates inventory record
  - Creates transaction
  - Confirms ownership
  - Updates quantities and costs
